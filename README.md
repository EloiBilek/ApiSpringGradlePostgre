# ApiSpringGradlePostgre
It 's a very simple example of an RestFull API, CRUD of User.
Using:
- Java 8
- Gradle 2.11
- Spring 4
- Hibernate 5
- DataSources/Resource Pools C3P0 0.9.1.2
- Tomcat 8
- Eclipse Java EE IDE - Version: Mars.1 Release (4.5.1)
